Fantrax

Fantasy football data analytics

In this project I analysed data obtained from the Fantrax English Premier League fantasy football website. The goal was to work in Python and earn common analytics and visualization tools such as Pandas, Seaborn, Matplotlib.

Besides learning the basics of Java I also learnt:

  • Cleaning data
  • Creating, appending, splitting and merging databases
  • Using various graphs to visualize data and draw conclusions
  • Styling graphs

The project was undertaken at the start of the football season so the data was not fully comprehensive. Source code.
