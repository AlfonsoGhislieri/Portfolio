Task Divider

A Java android application

I created an Android Application called "Task Divider" which divides tasks for users. This was achieved using Android Studio and Java. I used it as an opportunity to learn a language (Java) and create my first ever application.

Besides learning the basics of Java I also learnt:

  • Using xml to create the UI
  • Creating and implementing buttons
  • Incorporating error messages to combat possible user issues
  • Creating multiple fragments and using arrayAdapters to change from one to another
  • Using Bundles to transfer data across fragments

Although I am happy with the project as a whole it could still be worked on and improved. Things to consider implementing could be:

  • Adding weight to inputted tasks for better balancing
  • Adding minimum and max assigned tasks per user depending on total users
